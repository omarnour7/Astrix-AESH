# GeoRadar — Simulation Assumptions & Limitations

**Team Astrix · AESH 2026**  
This document discloses all assumptions underlying GeoRadar's simulated results,
as required by AESH submission standards.

---

## Signal Simulation

| Assumption | Value / Source | Justification |
|---|---|---|
| Frequency range | 200 MHz – 1 GHz | Optimal for 0–10 m penetration in dry soil (literature) |
| Max penetration depth | 10 m | Conservative estimate for arid soil at 200 MHz |
| Dielectric constants (εr) | Dry sand: 3.0 · Moist soil: 10.0 · Water: 25.0 · Rock: 6.5 | Standard GPR literature values (Daniels, 2004) |
| Conductivities (σ) | See `signal_gen.py` DEFAULT_PROFILES | Derived from NASA/ESA open soil datasets |
| Noise model | AWGN at 20 dB SNR | Idealized; real terrain introduces clutter not modelled |
| Soil layers | Modelled as homogeneous horizontal slabs | Real geology is heterogeneous; 3D scattering not modelled |
| Antenna effects | Not modelled | Real beam patterns affect signal shape |

## Duty-Cycle Power Model

| Assumption | Value | Note |
|---|---|---|
| Radar TX power (active) | 0.5 W | Low-power GPR module estimate |
| Radar RX power (active) | 0.1 W | Estimate |
| Radar idle power | 0.02 W | Standby mode |
| Entropy sensor power | 0.005 W | Always-on environmental sensor |
| Conventional GPR reference | 2.0 W | Literature value for portable GPR systems |
| Water-zone fraction in terrain | 30% | Simulation parameter; varies by region |
| Entropy threshold | 3.5 bits | Tuned on synthetic data; requires field calibration |
| Power reduction result (70%) | Simulated estimate | Real reduction depends on terrain and threshold tuning |

## CNN Classifier

| Assumption | Value | Note |
|---|---|---|
| Training data | 100% synthetic | Generated by `signal_gen.py`; no real survey data used |
| Target accuracy | >85% | Achievable on synthetic test set; real-world may differ |
| Class balance | Approximately equal (tunable) | Real survey data is imbalanced toward dry soil |
| Generalization | Not validated on real GPR data | Field validation is a stated future step |

## What These Results Prove

- GeoRadar demonstrates a technically coherent pipeline from FMCW signal simulation through CNN classification to interactive visualization.
- The 70% power reduction is a **modelled estimate** that follows directly from the duty-cycle logic — it is not a hardware measurement.
- The CNN accuracy is measured on a **held-out synthetic test set** from the same distribution as training data — not on real-world GPR surveys.
- All results are reproducible from the submitted repository files.
